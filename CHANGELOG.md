# Changelog

## 0.1.0 — 2026-01-20
- Initial open-source release.
- ProofPack format: `run.json`, `updates.jsonl` (row-chained), checkpoint + artifact SHA256 binding.
- Replayable PAC certificate from raw artifacts (anytime-valid α-spending via Hoeffding bounds).
- Optional Ed25519 manifest signing (PyNaCl).
- CartPole PPO demo + independent verifier + tamper demo.
